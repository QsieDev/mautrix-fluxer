# mautrix-fluxer
A Matrix-Fluxer puppeting bridge based on [fluxergo](https://github.com/qsiedev/fluxergo),
forked from [mautrix-discord](https://github.com/mautrix/discord) and
[discordgo](https://github.com/bwmarrin/discordgo).

## Documentation
This bridge uses the same [mautrix-go] bridge framework as mautrix-discord, so most of
[mautrix-discord's setup docs][docs.mau.fi] apply here too, aside from anything
Discord-API-specific (login, webhooks, etc.) which instead talks to Fluxer.

[mautrix-go]: https://github.com/mautrix/go
[docs.mau.fi]: https://docs.mau.fi/bridges/go/discord/index.html
